# Quote Book

A web app that displays random quotes in a 3D page-flipping book animation. Quotes are fetched live from the Hitokoto API on each page turn.

## Features

- 3D book opening and page-flipping animation built with pure CSS transforms
- Live quote fetching via async JavaScript
- Forward and back navigation with session history — no repeated API calls
- Copy quote to clipboard
- Responsive layout for mobile and desktop

## Tech Stack

- HTML / CSS / JavaScript
- CSS 3D transforms (perspective, rotateY, preserve-3d)
- Hitokoto API (v1.hitokoto.cn)

## Getting Started

1. Clone the repo
2. Open `1.html` in a browser
3. No build steps or dependencies required
